## PCB Notes

# Header CN2
- Pin 1: 3.3V?
- Pin 2: SWDIO
- Pin 3: SWCLK
- Pin 4: nRST
- Pin 5: GND

# Boot Mode
- BOOT_0: 0
- BOOT_1: 0

# JTAG to SWD
- SWDCK: TCK
- SWDIO: TMS
- nRST
